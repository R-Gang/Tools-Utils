# Tools-Utils

## 实用工具类

引入方式：

[![](https://jitpack.io/v/R-Gang/Tools-Utils.svg)](https://jitpack.io/#R-Gang/Tools-Utils)

    allprojects {
        repositories {
            ...
            maven { url 'https://jitpack.io' }
        }
    }

    dependencies {
         implementation 'com.github.R-Gang:Tools-Utils:latest.integration'
    }

## Usage

### 初始化实用工具

```
    initToolsUtils(this)
```
